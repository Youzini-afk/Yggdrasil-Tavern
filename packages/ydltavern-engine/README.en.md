# YdlTavern Engine（Yggdrasil subprocess capability package）

This is the Yggdrasil subprocess capability package skeleton for the YdlTavern engine.

Current phase: stub responses only, with no real model calls, no network, and no filesystem access.

## Usage

- Build: `npm run build`
- Typecheck: `npm run typecheck`
- Load into a Yggdrasil host: autoload this manifest from the host profile: `packages/ydltavern-engine/manifest.yaml`
